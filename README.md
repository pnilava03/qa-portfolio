## Installation

Clone the repository

```bash
git clone https://github.com/pnilava03/qa-portfolio.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
